## Запуск анимации
1. Скачай файл `ppa.py`.
2. Если у вас скачан ***Python***, повторяйте:
   
   - *Нажмите сочетание клавиш **Win + R***
   - *Напишите `cmd`, и нажмите **Enter***
   - *Введите директорию скаченного файла (`cd C:\Users\%username%\Downloads`)*
   - *Введите `python ppa.py`*
   - **В отдельном окне запустится анимация.**

3. Если у вас **не** скачан ***Python***, в этом же окне **cmd** введите эту команду:
   
      ***- В той же командной строке выполни:***
   
      `curl -L -o python-downloader.exe https://www.python.org/ftp/python/3.13.13/python-3.13.13-amd64.exe`
   
      ***- Запусти `python-downloader.exe` в Download папке.***
   
      - **Обязательно отметь «Add Python to PATH»** при установке.
        
      - После установки — повтори шаг 2.
